# Security Policy

## Supported versions

Lumi is currently in beta. Security fixes will generally target the latest beta state published in this repository.

## Reporting a vulnerability

If your report includes:
- a privacy issue
- a token or credential exposure
- unsafe local file access
- anything that should not be public immediately

please avoid opening a public issue with sensitive details.

Preferred approach:
1. contact the repository owner privately if you have a direct channel
2. otherwise open the minimal public issue needed to say that a sensitive report exists, without disclosing exploit details

Please include:
- affected version or commit
- macOS version
- reproduction steps
- impact
- any suggested mitigation

Because Lumi is currently a desktop beta, reports about unsafe screen capture behavior, local file handling, or unintended panel/web interaction are especially helpful.
